This is a JSON file so needs to be opened on a server.
MAMP works great so you can just copy the whole folder to you MAMP htdocs folder.
1. Under the repository name, click Clone or download.
2. In the Clone with HTTPs section, click copy button, or highlight the link, right click and select 'copy' to copy the clone URL for the repository.
3. Open the repository file and open the index.html file in a server browser.
